## TASK 1: Web Page Layout

Designed a page using html tags without using div 
  
The layout consists of a header, navigation menu, main content area, side content sections, and a footer.


## Layout Structure/ Features Implemented

The layout is divided into the following sections:

1. Header  
   - Positioned at the top.  
   - Usually contains the website title or logo.

2. Navigation Menu
   - Below the header.  
   - Used for site navigation links.

3. Main Content Area
   - Center of the page.  
   - Contains the primary content of the webpage.

4. Side Content Sections  
   - Located on the left and right of the main content.  
   - Can be used for ads, links, or additional information.

5. Footer  
   - Positioned at the bottom.  
   - Contains copyright, contact info, or other links.
