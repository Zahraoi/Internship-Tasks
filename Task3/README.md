# Submission Form UI (HTML & CSS)

This task is a front-end implementation of a **Submission Form UI** created using **HTML5 and CSS3**, based on a given design reference. The goal was to replicate the layout, structure, and visual style of the provided image as closely as possible.

---

##  Task Description

The page contains a two-column layout:

### Left Section
- Submission title
- Description text
- Application deadline notice

### Right Section
- User input fields (Name, Email, Organisation)
- Award selection using checkboxes
- Textareas for description & results
- Submission confirmation button

A footer logo is positioned at the **bottom-right corner** of the red background, exactly as shown in the reference design.

---

## Technologies Used
- HTML5
- CSS3 (Flexbox for layout)
- No frameworks or libraries used
