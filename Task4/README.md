This project is a simple HTML & CSS layout inspired by a football awards card design.

## Features

- Navigation bar with logo
- Hero section with title and description
- Rotated awards icon positioned at bottom-right
- "Best Clubs" section with 5 club icons in one row
- Text aligned under each icon using Flexbox
- Clean and simple CSS styling

---

##  Technologies Used

- HTML5
- CSS3 (Flexbox, positioning)

---

##  Folder Structure
- index.html: Main structure and content.
- style.css: Visual presentation
- screen2.png: Reference design asset.
- logo.png, icon_awards.png: Branding asset.

---
## How to Run

1. Download or clone the repository
2. Open index.html in your browser
3. Make sure the Images folder is in the same directory
