## Projects Overview

### Task 1: Three-Column Web Layout
**Description:**  
A semantic HTML5 layout implementing a classic three-column structure. This project focuses on proper use of Flexbox to manage layout alignment and spacing across different screen sizes.

**Technologies Used:**  
- HTML5  
- CSS3 (Flexbox)

---

### Task 2: Catchub Landing Page Replica
**Description:**  
A high-fidelity, responsive web replica of the Catchub mobile application landing page. The project emphasizes accurate UI reproduction, layout consistency, and responsive behavior using only HTML and CSS.

**Technologies Used:**  
- HTML5  
- CSS3  
- Media Queries  
- Responsive Design Techniques

---

### Task 3: Submission Form Design

**Description:**
A responsive replication of a specific award submission form design (screen1.png). Includes interactive form logic, word count validation, and a custom dark red aesthetic.

**Technologies Used:** 
- HTML5 
- CSS3
